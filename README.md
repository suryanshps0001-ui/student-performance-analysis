# Student Performance Analysis

This project analyzes student marks data using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Project Structure
- analysis.py : Main analysis script
- data/students.csv : Dataset

## What I Did
- Calculated average, highest, and lowest marks
- Visualized student performance using bar charts

## How to Run
1. Install libraries:
   pip install pandas matplotlib
2. Run:
   python analysis.py
